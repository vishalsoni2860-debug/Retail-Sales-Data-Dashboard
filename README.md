# Retail Sales Data w/ Power Bi

![Retail Sales Data Dashboard](/Image/Retail_Project_Image_1.png)

## Introduction   
This project presents an interactive Retail Sales Dashboard using real-world dataset of 2023 developed through Microsoft Power BI.
The dashboard analyzes retail transaction data to uncover insights related to **Sales performance, Customer behavior, Product category contribution,
Revenue trends,Customer segmentation.** The goal of this project is to transform raw transactional data into meaningful business insights using data modeling, DAX, and interactive visualizations.

## Skill Showcased

- **Data Transformation (ETL) with Power Query:** Cleaned, shaped, and prepared the raw data for analysis by handling blanks, changing data types, and creating new columns.

- **Implicit Measures:** Formulated measures to derive key insights and KPIs like `Total Sales` and `Average Order Value` etc.

- **Core Charts:** Utilized **Column, Bar, Line,** and to compare total sales and track trends over time.

- **KPI Indicators & Tables:** Used **Cards** to display key metrics and **Tables** to provide granular, sortable data.

- **Dashboard Design:** Designed an intuitive and visually appealing layout, exploring both common and uncommon chart types to best tell the data story.

- **Interactive Reporting:**

  -   **Slicers:** To dynamically filter the report by Month Name,Age Group,Product Category,Sales Segment etc.

  -  **Drill-Through:** To navigate from a high-level summary to a contextual, detailed view.

  ## Dashboard Overview

  ### Page 1: Sales Overview Dashboard

![Retail Sales Data Dashboard](/Image/Retail_Project_Image_1.png)
I used KPI Card visuals to display key metrics such as Total Sales, Total Transactions, Total Quantity, and Average Order Value, providing a quick snapshot of overall business performance. A Line Chart was implemented to analyze monthly trends in Total Sales and Total Number of Transactions, helping identify seasonal patterns and performance fluctuations. Additionally, a Tree Map was used to visualize sales contribution by Product Category, highlighting high-revenue segments, while a Stacked Bar Chart was created to compare total quantity sold across product categories, enabling clear identification of the highest-selling products by volume.


  ### Page 2: Customer & Segmentation Analysis

  ![Retail Sales Data Dashboard](/Image/Retail_Project_Image_2.png)

  I created an interactive view to analyze customer behavior and sales segmentation. I added slicers for Month Name, Gender, Age Group, Product Category, and Sales Segment to enable dynamic filtering across the report. An Age Group column was derived from the Age field, and conditional formatting was applied to highlight important segments. I have used a Matrix and various charts to visualize sales distribution and demographic performance. Additionally, a drill-through feature to Page 3 allows deeper product category analysis, enhancing interactivity and supporting data-driven insights.

  ### Page 3: Advanced Sales Analysis

  ![Retail Sales Data Dashboard](/Image/Retail_Project_Image_3.png)

  I used a Line and Clustered Column Chart to compare sales trends with category performance, enabling deeper time-based and segment-wise analysis. A Stacked Column Chart was implemented to evaluate category contributions and comparative performance across different dimensions. Additionally, a Table visual was included to present detailed transactional insights and rankings. The dashboard incorporates advanced DAX functions to calculate key metrics such as growth rates (MoM Growth %), revenue contribution, and customer analytics, enhancing the overall analytical depth of the report.

  ## Conclusion:

  This Retail Sales Dashboard project demonstrates my ability to transform raw data into actionable insights using Power BI. By applying data modeling, interactive visuals, drill-through features, and advanced DAX measures like MoM Growth, I built a dynamic reporting solution that helps track sales performance, analyze customer behavior, and support data-driven business decisions.
